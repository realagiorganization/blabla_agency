# Samples

## English

Short samples show how the agency loop changes shape without changing method.

| Track                | Intake                   | Recorded context                                           | Ships                                   | Public proof                           |
| -------------------- | ------------------------ | ---------------------------------------------------------- | --------------------------------------- | -------------------------------------- |
| 🌐 Site refresh      | rough site brief         | Zoom review, copy comments, current demo                   | updated site, Storybook stories, PDF    | Pages, Storybook, commit trail         |
| ⚙️ Internal tool     | loose ops flow           | operator calls, queue notes, failure logs                  | portal slice, automation, runbook       | demo build, docs, CI logs              |
| 🤖 Automation repair | broken script or process | error traces, chat clarifications, manual workaround video | patched pipeline, tests, monitor update | Actions, artifact manifest, live check |

### Common rule

The draft starts the job. The recorded conversation and the current demo make the next pass correct.

## Русский

Короткие примеры показывают, как меняется форма проекта без смены самого метода.

| Трек                     | Вход                          | Записанный контекст                                    | Что поставляется                                  | Публичное доказательство                 |
| ------------------------ | ----------------------------- | ------------------------------------------------------ | ------------------------------------------------- | ---------------------------------------- |
| 🌐 Обновление сайта      | грубый бриф сайта             | Zoom-ревью, комментарии к тексту, текущее демо         | обновленный сайт, Storybook-stories, PDF          | Pages, Storybook, цепочка коммитов       |
| ⚙️ Внутренний инструмент | свободное описание ops-потока | созвоны с операторами, заметки по очередям, логи сбоев | срез портала, автоматизация, runbook              | demo-билд, docs, CI-логи                 |
| 🤖 Починка автоматизации | сломанный скрипт или процесс  | error-трейсы, чатовые уточнения, видео ручного обхода  | исправленный пайплайн, тесты, обновление монитора | Actions, манифест артефактов, live-check |

### Общее правило

Черновик только запускает работу. Записанный разговор и текущее демо делают следующий проход точным.
